## 1. Scope

| # | 기능 | 설명 | 우선순위 |
|---|---|---|---|
| **M1** | StampLine 컴포넌트 분리 + 문서 상세 적용 | 인라인 → 재사용 컴포넌트, 문서 상세에도 표시 | P0 |
| **M2** | OrgTreePicker 기안 페이지 통합 | ApproverPicker 대체 옵션으로 OrgTreePicker 연동 | P0 |
| **M3** | 문서 이관 (transfer) | 관리자가 기안자를 다른 멤버로 변경 | P1 |
| **M4** | 보관기한 + 보안등급 메타데이터 | retention_period, security_level 컬럼 + UI | P1 |
| **M5** | admin 사이드바 네비게이션 정리 | 새로 추가된 라우트(audit, approval-templates 등) 네비 추가 | P1 |

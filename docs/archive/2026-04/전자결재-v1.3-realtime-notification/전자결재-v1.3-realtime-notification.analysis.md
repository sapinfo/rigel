|---|---|---|---|

| G3 | Inbox/ApprovalPanel 반응형 변경 미적용 | Medium | M22 scope — 추후 별도 커밋으로 추가 가능 |
| G4 | Edge Function SMTP 실제 발송 미구현 (console fallback) | Low | 프로덕션 배포 시 SMTP 환경변수로 활성화 |

## 3. Recommended Actions

### Immediate (M18 gate 전)

1. **G1**: `0037_approver_group_idx.sql` migration 추가
   ```sql
   CREATE INDEX IF NOT EXISTS as_approver_group_idx
     ON public.approval_steps (approver_user_id, group_order)
     WHERE status = 'pending';
   ```

2. **G7**: fn_submit_draft v3 form SELECT에 `FOR UPDATE` 추가 여부 결정
   - Option A: 0037에서 `CREATE OR REPLACE` 로 수정
   - Option B: SMB 컨텍스트에서 risk 수용 (v1.0/v1.1도 없었음)

### Documentation Update

3. **Design §7 업데이트**: `event.fetch + setContent` 방식 반영 (현재 설계서는 폐기된 `page.goto` 방식 기술)

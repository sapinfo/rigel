| **Problem** | 양식은 JSON 수동 편집, 결재선은 기안자가 매번 수동 구성 → 관리자 IT 의존도 높음 |

| **Core Value** | **"관리자 자립"** — 셀프서비스 완성, 고객 온보딩 비용 제거 |
